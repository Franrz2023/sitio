#  Batalla Robótica Capacitaciones

Sitio web oficial de **Batalla Robótica**, capacitación docente en robótica educativa con metodologías activas e integración curricular transversal.



## Características

- Landing page responsive con secciones de Inicio, Programas y Galería
- 3 programas por nivel educativo (Inicial, Primario, Secundario) con modal de detalles
- Efecto de profundidad/parallax en las tarjetas de programas (`data-depth`)
- Inscripción integrada vía formulario embebido (Tally)
- Loader animado de carga inicial
- Menú de navegación responsive con toggle mobile

##  Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- [Tally](https://tally.so) para formularios de inscripción
- Google Fonts (Inter)

## Estructura

```
├── index.html
├── style.css
├── script.js
└── assets/
```



##  Programas

| Nivel | Edad | Enfoque |
|---|---|---|
| Inicial | 3-5 años | Lógicas tangibles, sin pantallas |
| Primario | 6-11 años | STEAM, bloques visuales interactivos |
| Secundario | 12-17 años | Diseño 3D, programación textual, electrónica |
